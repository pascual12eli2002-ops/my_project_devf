#Proyecto final del Módulo 1

#Descripción
Este proyecto es una página web informativa sobre Programación con Python, creada como parte del Módulo 1 del programa DEV.F. Incluye distintos elementos de HTML como listas, imágenes, encabezados, navegación, artículos, spans, divs, enlaces y estructura semántica, con estilos para mejorar su presentación.

#Contenido
El proyecto contiene:
Página principal: index.html
Uso de etiquetas semánticas (header, nav, main, section, article, footer)
Listas ordenadas y desordenadas
Imágenes del lenguaje Python
Ejemplo de código
Estilos CSS integrados
Uso de div y span con estilos
Enlace a la documentación oficial de Python

#Próposito del proyecto
Este proyecto refleja mis capacidades aprendidas en el Módulo 1 del programa DEV.F, aplicando conocimientos de estructura HTML y organización visual de una página web.

#Instrucciones para ejecutar
1.-Descarga o clona este repositorio.
2.-Abre el archivo index.html en un navegador web.
3.-No requiere instalación adicional.
